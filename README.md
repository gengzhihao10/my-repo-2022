本项目仅为练习用，因此暂时不考虑微服务，由于需要加入中间件，因此将模块确定为下列几个模块。

​		

| 模块名称   | 描述                 |
| ---------- | -------------------- |
| boot       | 启动模块             |
| common     | 通用工具类和常量模块 |
| middleware | 中间件模块           |
| dal        | 数据库交互模块       |
| api        | 对外接口层模块       |
| present    | 表现层模块           |
| bussiness  | 业务层               |
